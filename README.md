## Installation

  

Install the project with pip , pipenv

  We are Using PIPENV as a package manager

```bash
cd zidship

pip install pipenv

pipenv shell

pipenv install

Python manage.py migrate

Python manage.py runserver
```

  

Then you will need to create your .env

file inside zidship directory
